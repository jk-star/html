# 📘 Chapter 12 – Meta Tags & SEO

## Q1. Meta Tag Kya Hota Hai?

- Meta Tag webpage ke baare me extra information provide karta hai.
- Ye information:
    - Browser ke liye
    - Search Engine ke liye
    - Social Media ke liye hoti hai.
- User ko webpage par directly dikhai nahi deti.

**Example**

`<meta charset="UTF-8">`

## Q2. Charset
- Browser ko batata hai ki webpage UTF-8 encoding use kar raha hai.

`<meta charset="UTF-8">`

**UTF-8 Kya Hai?**

Ek character encoding standard. Ye Hindi, English, Emoji 😊 ,Special Characters sabko support karta hai.

## Q3. Viewport
- Website ko mobile responsive banane me help karta hai.
- `width=device-width` Browser ko bolta hai Screen jitni width hai Utni width use karo.
- `initial-scale=1.0` Zoom level normal rakho.

`<meta name="viewport" content="width=device-width, initial-scale=1.0">`

## Q4. Description
- Google Search Result me description show ho sakta hai.
- Description SEO ke liye important hai.

`<meta name="description" content="Online Shopping Website">`

## Q5. Keywords
- Purane time me use hota tha.

`<meta name="keywords" content="HTML,CSS,JavaScript">`

- Google isse almost ignore karta hai.

## Q6. Robots

`<meta name="robots" content="index, follow">`

**Meaning**

**index**

- Google page ko index kar sakta hai.

**follow**

- Page ke links ko follow kar sakta hai.

**noindex**

- Google page ko search result me nahi dikhayega.

**nofollow**

- Links follow nahi karega.

## Q7. Author
- Page kisne banaya.

`<meta name="author" content="Jyoti Singh">`

## Q8. Favicon

`<link rel="icon" href="favicon.ico">`
