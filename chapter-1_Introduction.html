Q1. HTML Kya Hai?

  • HTML (HyperText Markup Language) ek Markup Language hai jo webpage ka structure banane ke liye use hoti hai.

  • HTML website ka skeleton hota hai.

  • Ye browser ko batata hai ki page par:

  • Heading kahan hogi

  • Paragraph kahan hoga

  • Image kahan hogi

  • Form kahan hoga

  • Button kahan hoga

HTML design nahi karta, sirf structure banata hai.

Q2. HTML Programming Language Hai?

  • Nahi, HTML programming language nahi hai.

  • Ye Markup Language hai.

Programming language me hota hai:

  • Variables

  • Loops

  • Functions

  • Conditions (if-else)

  • Logic

Example:

  • PHP

  • JavaScript

  • Java

  • Python

Q.4 Markup Language kya hota hai

Markup language sirf content ko describe karti hai.

Example: 

<h1>Welcome</h1>
<p>Hello World</p>

  • Isme koi logic nahi hai.

Q.5 HyperText Kya Hai?

  • HyperText ka matlab hai:

  • Ek page se dusre page par link ke through jaana.

Example: 

<a>  href="href="about.html">About Us </a>

  • Is link par click karte hi dusra page open ho jayega.

  • Isi concept ko HyperText kehte hain.

Q.6 Browser HTML Ko Kaise Read Karta Hai?

Jab browser HTML file open karta hai:

  • HTML ko line by line read karta hai.
  • Tags ko Analyze karke process karta hai.
  • DOM (Document Object Model) banata hai.
  • CSS apply karta hai.
  • JavaScript execute karta hai.
  • User ko webpage dikhata hai.

Practical Example 

<!DOCTYPE html>
<html>
<head>
<title>My Website</title>
</head>
<body>
<h1>Welcome</h1>
<p>This is my first HTML page.</p>
</body>
</html>
Browser Output 
 Welcome
This is my first HTML page.
